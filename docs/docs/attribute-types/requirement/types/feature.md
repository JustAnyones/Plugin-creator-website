# FEATURE

Used to specify a feature that has to be bought for the requirement to be fulfilled.

## Example

The following example requires the player to own the Christmas feature.

```json
"requirements": [
  {
    "type": "FEATURE",
    "id": "$feature_christmas00"
  }
]
```
